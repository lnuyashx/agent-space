# GitHub Labels

Recommended labels for repository setup:

## Areas

- `area:renderer` - rendering, object layer, PixiJS, sprites, depth sorting
- `area:interaction` - hitAreas, hover/click, movement, navigation, collision
- `area:data` - item catalog, inventory, scene snapshots, persistence
- `area:qa-docs` - tests, verification, README, architecture docs
- `area:product` - gameplay/product design, monetization assumptions, roadmap

## Types

- `type:feature`
- `type:bug`
- `type:tech-debt`
- `type:docs`
- `type:research`

## Status

- `status:ready`
- `status:claimed`
- `status:blocked`
- `status:needs-review`

## Priority

- `priority:p0`
- `priority:p1`
- `priority:p2`
